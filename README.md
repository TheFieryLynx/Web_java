# Описание страниц
-------------
*****
## Для незарегистрированных пользователей
### Верхнее меню:
1. Кнопка "Root page" (переход на главную страницу "Root page") - на всех страницах
2. Кнопка "Registration" (Переход на страницу регистрации)
3. Кнопка "Log In" (переход на страницу входа)
##### Дополнительные страницы:
4. Страница книги
5. (функциональность) справа на главной странице фильтрация по названию книги, автору и жанру.

### #1. Root page:
Является главной страницей. Список всех книг. Для каждой книги: название, автор, жанр, цена.
Название книги - ссылка с переходом на страницу книги

#### 2. Registration:
Страница регистрации с формой для заполнения. 

#### 3. Log in:
Страница входа пользователя на сайт под своим логином и паролем.

#### 4. Страница книги:
Страница со всей информацией о книге: название, автор, жанр, издательство, год издания. количество страниц, тип обложки, цена книги, количество на складе.


## Для зарегистрированных пользователей
### Верхнее меню:
1. Кнопка "Root page" (переход на главную страницу "Root page") - на всех страницах
2. Кнопка "Personal area" (Переход на страницу с личной информацией)
3. Кнопка "Orders" (переход на страницу с заказами пользователя)
4. Кнопка "Log Out" (переход на главную страницу незарегистрированного пользователя)
##### Дополнительные страницы:
5. Страница книги с кнопкой покупки.
6. Страница проверки заказа с кнопкой редактирования и покупки.
7. Страница редактирования заказа.
8. Страница с личной информацией о пользователе.
9. Страница редактирования личной информации
10. Страница с заказами пользователя
11. (функциональность) справа на главной странице фильтрация по названию книги, автору и жанру.

### #1. Root page:
Является главной страницей. Список всех книг. Для каждой книги: название, автор, жанр, цена.
Название книги - ссылка с переходом на страницу книги

#### 2. Personal area:
Страница с личной информацией пользователя: логин, пароль, ФИО, адрес проживания, телефон, почта. Кнопка "Edit" -> переход на страницу редактирования личной информации.

#### 3. Orders:
Страница пользователя со всеми сделанными когда-либо заказами.

#### 4. Log Out:
Переход на главную страницу незарегистрированного пользователя

#### 5. Страница книги с кнопкой покупки:
Страница со всей информацией о книге: название, автор, жанр, издательство, год издания. количество страниц, тип обложки, цена книги, количество на складе. Кнопка покупки книги.

#### 6. Страница проверки заказа с кнопкой редактирования и покупки:
Страница с информацией о покупаемой книге и кнопками редактирования и оформления заказа (покупки)

#### 7. Страница редактирования заказа.
Возможность изменить информацию о пользователе для заказа без сохранения в бд.

#### 8. Страница с личной информацией о пользователе.
Личная информация: логин, пароль, ФИО, адрес проживания, телефон, почта. Кнопка "Edit" -> переход на страницу редактирования личной информации.

#### 9. Страница редактирования личной информации
Редактирование полей с данными пользователя из п.8

#### 10. Страница с заказами пользователя
Отображение таблицы со всеми заказами

### Для администратора
### Верхнее меню:
1. Кнопка "Root admin page" 
2. Кнопка "Orders"
3. Кнопка "Customers"
4. Кнопка "Add book" 
5. Кнопка "Add customers" 
6. Кнопка "Log Out" 

#### 1. Root admin page: 
Является главной страницей. Список всех книг. Для каждой книги: название, автор, жанр, цена.
Название книги - ссылка с переходом на страницу книги

#### 2. Orders: 
Страница для администратора со всеми заказами от пользователей. Возможность изменения каждого заказа.

#### 3. Customers:
Список всех пользователей по логинам - со ссылками на личные кабинеты.

#### 4. Кнопка "Add book" 
Кнопка добавления книги - появляется страница с полями для заполнения

#### 5. Кнопка "Add customers" 
Кнопка добавления пользователей - появляется страница с полями для заполнения

#### 6. Кнопка "Log Out" 
Переход на главную страницу незарегистрированного пользователя 

Сценарии использования
-------------
### 1. Незарегистрированные пользователи
- Просмотр каталога книг на странице "Книги":
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг.
- Возможность поиска на странице "Поиск"
- - С любой страницы: кнопка "Поиск" в верхнем меню -> переход на страницу поиска.
- Возможность зарегистрироваться на сайте на странице "Регистрация"
- - С любой страницы: кнопка "Регистрация" в верхнем меню -> переход на страницу "Регистрация".
  
### 2. Зарегистрированные пользователи
- Просмотр каталога книг на странице "Книги"
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг.
- Возможность поиска на странице "Поиск"
- - С любой страницы: кнопка "Поиск" в верхнем меню -> переход на страницу поиска.
- Просмотр личного кабинета
- - С любой страницы: кнопка "Личный кабинет" в верхнем меню -> переход на страницу личного кабинета.
- Редактирование Личного кабинета
- - С любой страницы: кнопка "Личный кабинет" в верхнем меню -> переход на страницу личного кабинета -> кнопка "Редактировать" -> переход на страницу редактирования личного кабинета
- Возможность добавлять книги в определенном количестве в корзину
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг -> Кнопка "Добавить в корзину" -> переход на страницу редактирования количества добавляемых книг
- Просмотр корзины, изменение количества книг в ней. 
- - С любой страницы: кнопка "Корзина" -> переход на страницу "Корзина" -> кнопка "Редактировать" -> переход на страницу редактирования страницы.
- Переход на страницу "Оформление заказа"
- - С любой страницы: кнопка "Корзина" -> переход на страницу "Корзина" -> кнопка "Оформить заказ" -> Переход на страницу "Оформление заказа"

### 3. Администраторы
- Просмотр всех заказов от пользователей
- - С любой страницы: кнопка "Заказы" -> переход на страницу "Заказы"
- Редактирование заказов
- - С любой страницы: кнопка "Заказы" -> переход на страницу "Заказы" -> кнопка "Редактировать" -> переход на страницу редактирования заказов.
- Просмотр личных кабинетов пользователей
- - С любой страницы: кнопка "Пользователи" -> переход на страницу "Пользователи" -> переход на страницу личного кабинета пользователя по нажатию на пользователя -> кнопка "Редактировать" -> переход на страницу редактирования страницы.
- Добавление новых книг
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг -> кнопка "Добавить" -> переход на страницу добавления книги
- Удаление книг
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг -> кнопка "Удалить"
- Редактирование информации о книгах
- - С любой страницы: кнопка "Книги" в верхнем меню -> переход на страницу со списком книг -> кнопка "Редактировать" -> переход на страницу редактирования книг.
